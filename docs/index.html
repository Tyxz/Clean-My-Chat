MessageNeedsToBeRemoved (messageType, fromName, text, isFromCustomerService, fromDisplayName)
Will check a chat event formatter, if it should be removed based on the set filter

Parameters:

  • messageType number channel
  • fromName string name of the account it is send from
  • text string message
  • isFromCustomerService boolean if the message is from a customer service agent
  • fromDisplayName string name of the character it is send from

Returns:

    boolean if the message should be filtered out
